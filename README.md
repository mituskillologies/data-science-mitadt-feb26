## 📌 About The Project
This repository contains the programming materials, scripts, and Jupyter Notebooks used during the **"Data Analysis using Python"** program conducted at **MIT ADT University, Pune** in February 2026. 

The course covers fundamental Python concepts, Object-Oriented Programming (OOP), Data Structures, and popular Data Science libraries like NumPy and Pandas, culminating in feature engineering and practical data analysis.

## 📂 Repository Contents

The repository is structured progressively from basic Python programming to advanced data science concepts:

* **`01Basic_Feb24.ipynb`**: Introduction to Python basics and syntax.
* **`02Decision_making_Feb24.ipynb`**: Conditional statements and decision-making in Python.
* **`03DataStructures_Feb25.ipynb`**: Deep dive into Python's core data structures (Lists, Tuples, Dictionaries, Sets).
* **`04Functions_and_Modules_feb25.ipynb`**: Creating and utilizing functions and modular programming.
* **`05 mod1.py`**: Example Python module script.
* **`06module_use.ipynb`**: Implementing and utilizing custom modules in Jupyter.
* **`07 OOP_feb25.ipynb`**: Object-Oriented Programming concepts in Python.
* **`08 Numpy_feb25.ipynb`**: Numerical computing and array manipulation using the `NumPy` library.
* **`09 Pandas_feb25.ipynb`**: Introduction to data manipulation and analysis using `Pandas`.
* **`10 pandas_da_feb26.ipynb`**: Practical Data Analysis tasks utilizing Pandas.
* **`11 feature_engg_feb26.ipynb`**: Advanced feature engineering techniques for preparing data for machine learning models.
